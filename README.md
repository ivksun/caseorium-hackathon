# Caseorium — AI Case Study Pipeline

YouTube-ссылка (или текстовая фактура) → автономный мульти-агентный пайплайн → черновик кейса в WordPress.

## Агенты

1. **Транскрибер** — видео → чистая расшифровка (Deepgram / YouTube captions)
2. **Аналитик** — разбор транскрипта + слайдов (multimodal) + обогащение через веб-поиск
3. **Райтер** — кейс по структуре Generation AI, инфостиль, SEO
4. **Редактор** — валидация фактов, антинейросетевая чистка, чек-лист
5. **Верстальщик** — публикация черновика в WordPress с ACF-полями и таксономиями

## Структура

```
agents/          — код агентов (Claude Agent SDK / Claude API)
tools/           — утилиты (транскрипция, слайды, WP publish)
engine/          — промпты, правила, пайплайн-документация
knowledge/       — база знаний: продукт, аудитория, эксперты
cases/examples/  — референсные кейсы
config/          — конфигурация пайплайна (WP, модели, структура кейса)
references/      — стайлгайд, ToV
```

## Quickstart

```bash
# 1. Настроить .env
cp .env.example .env
# заполнить: ANTHROPIC_API_KEY, DEEPGRAM_API_KEY, WP_URL, WP_USER, WP_APP_PASSWORD

# 2. Установить зависимости
pip install -r requirements.txt

# 3. Запустить пайплайн
python main.py --youtube "https://youtube.com/watch?v=..." --speaker "Иванов"
```

## Метрики

| Было | Стало |
|------|-------|
| 3-4 часа редактора + неделя ожидания дизайна | 15-30 минут автономной работы |
| Ручная верстка в WP | Автопубликация черновика |
| 0 масштабируемость | Батч-обработка 30-50 кейсов |
